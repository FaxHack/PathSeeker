Features

Advanced Chunk Detection

Identify new and modified chunks with precision, perfect for base hunting.

Base Finder

Sophisticated algorithms to detect potential base locations and player activity.

Storage Looter

Efficiently manage and search through storage systems with enhanced visibility.

StackedMinecartDetector

Detects for hidden stacked spawner is in the area.

GrimDuraFirework

A clever fix for fireworks issues on the Mio client when using Grim Durability ElytraFly with a chestplate. Fly smoothly without a hitch!

ESP Features

Comprehensive ESP system including PotESP, HoleESP, and MobGearESP.

Latest Releases

Latest Release

v1.0.6

What's New:

  • BetterStashFinder: An upgraded stash finder with additional filtering options.
  • AutoEnchanter: Automates enchanting at anvils, saving time.
  • GrimEfly: Vanilla efly using a chestplate so that elytra does not use durability.
  • Trident Dupe: Enables trident duplication (note: patched on most servers).
  • AutoFirework: Automatically triggers firework use for extra flair.

Current Features:

    Utility:
  • ActivatedSpawnerDetector: Detects activated spawners nearby
  • SignHistorian: Record and restore broken or modified signs
  • ElytraSwap: Automatic elytra management
  • NoJumpDelay: Removes delay between jumps
  • GrimEfly: Vanilla efly using a chestplate so that elytra does not use durability.
    • Exploration:
  • Base Finder: Advanced base detection for anarchy servers
  • NewChunks: Identify newly generated chunks
  • PortalPatternFinder: Locate broken/removed Nether Portals
  • CaveDisturbanceDetector: Detect minor cave disruptions
  • TrailFollower: Follows trails using pitch40 or baritone (Requires XaeroPlus)
  • Pitch40Util: Height gain and velocity management
    • Renders:
  • Hole, Tunnel, and Stair ESP
  • PotESP: Decorated Pot tracking
  • MobGearESP: Player equipment detection on mobs
  • StackedMinecartDetector: Stacked spawner detection
  • EntityClusterESP: Entity cluster center point highlighting
  • DroppedItemESP: Important item highlighting
    • Automation:
  • StorageLooter: Automatically loots containers and organizes items
  • Trident Dupe: Enables trident duplication (note: patched on most servers).
  • AutoFirework: Automatically triggers firework use for extra flair.
  • AutoEnchanter: Automates enchanting at anvils, saving time.

Known Issues:

  • Minor rendering glitches during dimension transitions
  • Rare server-side compatibility issues with certain chunk detection methods
